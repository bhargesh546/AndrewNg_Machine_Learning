# Terminology

**Training Set:** Data used to train a model

**x:** "input" / feature variable

**y:** "output" / target variable

**m:** number of training examples

**(x, y):** single training example

**(x<sup>(i)</sup>, y<sup>(i)</sup>):** i<sup>th</sup> training example from 1, 2, .... m

**$\hat{y}$:** estimated y using the model/function

**Straight line function:**  Also known as linear function with one variable OR univariate linear regression
<center>f<sub>(w,b)</sub>(x) = wx + b</center>

<br>

- **w, b** = parameters that can be adjusted to improve the model, aka coefficients or weights
